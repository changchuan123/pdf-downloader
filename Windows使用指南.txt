PDF下载器 - Windows使用指南
================================

## 方案一：Python环境运行（推荐）

### 1. 安装Python
- 访问 https://www.python.org/downloads/
- 下载Python 3.8或更高版本
- 安装时勾选"Add Python to PATH"

### 2. 准备文件
将以下文件复制到Windows电脑的同一文件夹：
- pdf_downloader.py （主程序）
- requirements.txt （依赖清单）
- test_urls.xlsx 或 urls.xlsx （URL文件）
- fp.txt （可选，自定义文件名）

### 3. 安装依赖
打开命令提示符(cmd)，进入文件夹，执行：
```
pip install -r requirements.txt
```

### 4. 运行程序
```
python pdf_downloader.py
```

## 方案二：Windows EXE打包

### 前提条件
需要在Windows电脑上进行打包操作

### 打包步骤
1. 按方案一完成Python环境配置
2. 安装PyInstaller：
   ```
   pip install pyinstaller
   ```
3. 创建Windows版spec文件（pdf_downloader_win.spec）
4. 执行打包命令：
   ```
   pyinstaller pdf_downloader_win.spec --clean
   ```

## Excel文件格式要求
- 文件名：urls.xlsx 或 test_urls.xlsx
- 第一列：PDF下载链接
- 第二列：文件名（可选）
- 保存格式：Excel工作簿(.xlsx)

## 运行结果
- 下载文件保存在：pdf_dl文件夹
- 显示下载进度和统计信息
- 自动处理重名文件

## 常见问题
1. 如果提示"python不是内部命令"：
   - 重新安装Python并勾选"Add to PATH"
   - 或使用完整路径运行

2. 如果pip安装失败：
   - 尝试使用：python -m pip install -r requirements.txt
   - 检查网络连接

3. 如果下载失败：
   - 检查URL是否有效
   - 确认网络连接正常
   - 部分网站可能有访问限制
